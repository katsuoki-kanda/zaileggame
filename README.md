# 顔・手指・体のいずれかを認識して遊ぶゲーム

ブラウザで遊べるゲームです。  
Webカメラで顔・手指・体のいずれかを認識し、MediaPipeを使ってプレイします。

## 遊び方

- ブラウザでゲームページを開きます。
- カメラへのアクセスを許可します。


## 使用している技術

- JavaScript / HTML / CSS
- Google MediaPipe

このゲームは Google MediaPipe を利用しています。  
MediaPipe は Apache License 2.0 のもとで提供されています。
